# Run Order

This document gives a practical order for reproducing the public-data analyses.

## 0. Environment

Run:

```bash
Rscript scripts/00_setup/00_install_packages.R
Rscript scripts/00_setup/01_check_environment.R
```

The study workflows were run using R 4.5.2. Later R versions should also work if the required CRAN and Bioconductor packages are available.

## 1. DNA Methylation Pipelines

Run each pipeline from its pipeline root folder.

```bash
cd pipelines/blood_methylation
Rscript scripts/00_run_all_blood_methylation.R

cd ../brain_methylation
Rscript scripts/00_run_all_brain_methylation.R

cd ../placenta_lcl_methylation
Rscript scripts/00_run_all_placenta_lcl_methylation.R
```

Large WGBS files may be downloaded from public repositories or staged locally. Relevant environment variables include `BLOOD_WGBS_DOWNLOAD_REPORTS`, `BRAIN_WGBS_SOURCE_CACHE`, and equivalent source-cache settings noted in the scripts.

## 2. Gene Expression Pipelines

Run:

```bash
cd pipelines/blood_expression
Rscript scripts/00_run_all_blood_expression.R

cd ../brain_expression
Rscript scripts/00_run_all_brain_expression.R

cd ../placenta_lcl_expression
Rscript scripts/00_run_all_placenta_lcl_expression.R
```

Useful optional source-cache variables include `BLOOD_EXPRESSION_SOURCE_CACHE`, `BRAIN_EXPRESSION_PUBLIC_SOURCE_CACHE`, and `PLACENTA_LCL_EXPRESSION_PUBLIC_SOURCE_CACHE`.

## 3. GSE36315 Custom-Annotation Sensitivity

Run only after the main brain-expression pipeline and custom probe remapping inputs are available.

```bash
cd pipelines/gse36315_custom_probe_remapping
Rscript scripts/run_GPL15314_custom_probe_remapping_bowtie.R
Rscript scripts/assess_GSE36315_gene_level_recoverability.R

cd ../gse36315_brain_expression_sensitivity
Rscript scripts/run_gse36315_custom_annotation_brain_expression_sensitivity.R
```

These outputs are sensitivity-only and should not replace the primary brain expression model.

## 4. Meta-Analysis Model Fitting With Metafor

From the package root:

```bash
Rscript scripts/04_meta_analysis/00_run_all_meta_analysis_models.R
```

This wrapper runs the DNA methylation and gene-expression meta-analysis fitting scripts separately:

```bash
Rscript scripts/04_meta_analysis/01_fit_methylation_models_metafor.R
Rscript scripts/04_meta_analysis/02_fit_expression_models_metafor.R
```

The input to this step is the dataset-level Hedges' g table generated by the branch-specific processing pipelines. The output is a set of gene-level DerSimonian-Laird random-effects estimates, FDR values, modified Knapp-Hartung interval classifications, and model summaries.

## 5. Cross-Omic Convergence And Pathway Enrichment

Run:

```bash
Rscript scripts/05_convergence_pathway/run_cross_omic_pathway_enrichment.R
Rscript scripts/05_convergence_pathway/postprocess_pathway_enrichment_outputs.R
Rscript scripts/05_convergence_pathway/run_continuous_cross_omic_convergence_enrichment.R
Rscript scripts/05_convergence_pathway/run_brain_subtissue_cross_omic_convergence.R
```

## 6. Figures

Run:

```bash
Rscript scripts/06_figures/build_dataset_search_flow_diagram_prisma_style.R
Rscript scripts/06_figures/build_enhanced_pathway_figures.R
```

## Reproducibility Notes

Some public repository files are too large to store in a code repository. The scripts either download public sources or allow the user to point to a local folder containing the downloaded public source files.

All paths should be set through environment variables or relative to the project root. No personal machine path should be required.
